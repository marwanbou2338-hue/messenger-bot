# 🤖 بوت المسنجر للمجموعات

بوت فيسبوك مسنجر متكامل للمجموعات مع دعم الأوامر، المحرك، والقفل.

---

## 🚀 طريقة التشغيل

### 1. تثبيت المتطلبات

```bash
cd artifacts/messenger-bot
npm install
```

### 2. إعداد الـ AppState (الكوكيز)

ضع ملف `appstate.json` في مجلد البوت. يمكنك الحصول عليه باستخدام:
- إضافة **c3c-ufc-utility** في متصفح Chrome
- أو أداة **fbstate** في Termux

الملف يجب أن يكون بهذا الشكل:
```json
[
  { "key": "c_user", "value": "...", "domain": ".facebook.com", ... },
  ...
]
```

### 3. إعداد الـ Config

افتح `config.json` وعدّل:
```json
{
  "prefix": "/",
  "superAdminID": "ضع_ايدك_هنا"
}
```

### 4. تشغيل البوت

```bash
npm start
```

## 📋 الأوامر

| الأمر | الوصف | الصلاحية |
|-------|--------|----------|
| `/اوامر` | عرض قائمة الأوامر | الجميع |
| `/سيرفر` أو `/ابتيم` | معلومات البوت والسيرفر | الجميع |
| `/محرك` | التحكم في محرك الرسائل | أدمن |
| `/رفع` | رفع شخص إلى أدمن | سوبر أدمن فقط |
| `/اخفاض` | إنزال شخص من أدمن | سوبر أدمن فقط |
| `/قفل` | قفل البوت في الغروب | أدمن |

---

## ⚙️ أوامر المحرك

```
/محرك تشغيل         — تشغيل المحرك في الغروب الحالي
/محرك إيقاف         — إيقاف المحرك
/محرك رسالة [نص]    — تحديد رسالة المحرك
/محرك وقت [ثواني]   — تحديد وقت الإرسال (بالثواني)
/محرك الذكي تشغيل   — إرسال فقط عند وجود نشاط
/محرك الذكي إيقاف   — إرسال بشكل منتظم
```

## 🔒 أوامر القفل

```
/قفل الذكي تشغيل    — قفل البوت في جميع الغروبات
/قفل الذكي إيقاف    — رفع القفل عن جميع الغروبات
/قفل عادي تشغيل     — قفل البوت في هذا الغروب فقط
/قفل عادي إيقاف     — رفع القفل عن هذا الغروب
```

## 🔐 الصلاحيات

| الصلاحية | الوصف |
|---------|--------|
| **superadmin** | مطور البوت، لا يمكن تغيير صلاحياته |
| **admin** | أدمن البوت، يمكن رفعه وإنزاله من قبل superadmin |
| **user** | مستخدم عادي |

## 🔄 تجديد الكوكيز تلقائياً

البوت يحفظ الـ appstate تلقائياً كل 30 دقيقة في ملف `appstate.json`.

## 📁 هيكل الملفات

```
messenger-bot/
├── config.json          ← إعدادات البوت
├── appstate.json        ← كوكيز الحساب
├── data/
│   ├── admins.json      ← قائمة الأدمنز
│   ├── engine.json      ← إعدادات المحرك
│   └── lock.json        ← إعدادات القفل
└── src/
    ├── index.js         ← الملف الرئيسي
    ├── commands/        ← ملفات الأوامر
    └── utils/           ← الأدوات المساعدة
```
